# 과제 안내  

## 요구 사항  

weather_stations.csv는 `지역;온도` 의 형식을 가진 파일입니다.    
해당 파일을 파싱하여 간단한 기술 통계량을 계산하려 합니다.  

요구 사항은 다음과 같습니다.  
1. 지역별로 `최소/평균/최대 온도`를 계산해야 합니다.  
2. 결과는 관측소 이름 알파벳순으로 정렬해 출력해야 합니다.  
3. 결과의 값은 소숫점 1자리까지만 출력하도록 합니다.
